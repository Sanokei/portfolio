# Context

## Codebase Overview
This is a 3D museum gallery portfolio using JavaScript and Three.js (judging from scene.js, wall.js, etc.).

## Key Files
- `index.html`: Main HTML entry point.
- `main.css`: Main styling.
- `js/main.js`: Main application entry point.
- `js/scene.js`: 3D Scene setup.
- `js/scroll.js`: Scrolling logic.
- `js/wall.js`: Wall rendering logic.
- `js/layout.js`: Layout and spacing logic.
- `js/projects.js`: Project display logic.
- `js/plaque.js`: Plaque drawing and rendering logic.
- `js/environment.js`: Lighting, atmosphere, floor, etc.
- `js/decals.js`: Decorative elements.
- `js/interactions.js`: Click/hover interactions.
